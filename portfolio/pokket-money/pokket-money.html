Angular | Angular Material | Interacting with 3rd Party API

I developed Pokket Money after searching for a public/free API I could use to allow users to search and display information more easily. Pokket Money uses the OpenFEC API provided by the Federal Election Commission of the USA. After reviewing the API documentation for a project idea, I came up with this site to allow people to search for candidates at the federal level and see how and who is funding their campaigns. Disclaimer: All of the information displayed within the application is provided by the OpenFEC endpoint. This information is personal as it includes names, professions and addresses of contributors. With that said, I did not deploy this to the public as it could be used maliciously. However, I do have the project files on my GitHub page.
